title: "Utility Commands"
description: Miscellaneous bot utilities.
---

#### `/ping`

Check bot latency in milliseconds.

> **You:** `/ping`
>
> **Bot:** Pong! `42ms`

#### `/permissions` `<user>`

Display a user's Discord permission flags. Response is ephemeral (only you can see it).

#### `/help`

List all available commands grouped by category. Commands requiring permissions you don't have are hidden.
